Reading Times From a File

There are two different ways of automatically getting the chapter times entered into the list. The first method is by importing the times from an input file as described in Input Files, or from an internet lookup.

Note that the imported times will either be appended to the list of existing times or replace the existing list, depending on the user’s “Append” setting.
